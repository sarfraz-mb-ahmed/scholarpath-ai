# ScholarPath AI

A scholarship discovery and SOP generation platform designed specifically for Pakistani students.

## The Problem
Pakistani students often face fragmented scholarship data and lack the resources to write high-quality Statements of Purpose. ScholarPath AI solves this by centralizing verified scholarship data and using AI to draft professional, personalized SOPs.

## Key Features
- **Verified Search:** A curated scholarship database specifically for Pakistani nationals.
- **AI SOP Generator:** Secure, backend-processed AI drafts that are tailored to the student's specific academic profile and target scholarship.

- **Frontend:** React 18, Vite, Tailwind CSS
- **Backend:** Node.js, Express 4
- **AI Integration:** Google Gemini 1.5 Flash via OpenRouter
